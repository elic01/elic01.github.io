## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/elic01/elic01.github.io.git
cd elic01.github.io
```

### 2. Open Locally
Simply open `index.html` in your browser:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Or just double-click index.html
```

### 3. Set Up EmailJS (Optional but Recommended)
Follow the detailed guide in [SETUP_EMAILJS.md](SETUP_EMAILJS.md)
